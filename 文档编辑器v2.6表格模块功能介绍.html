高级特性与生产力工具

多选批量模式

单击“多选批量操作”按钮进入多选状态，随意点选单元格（支持跨行跨列），可批量进行：

  • 统一对齐方式
  • 统一字号 / 文字颜色 / 粗斜体
  • 批量清除格式

选中单元格有明显的蓝色外框 + 浅蓝背景，操作后自动退出多选模式。

撤销 / 重做 (完整历史)

内置记录栈，自动保存表格每次结构或内容变动（MutationObserver 监听）。支持快捷键：

  • Ctrl+Z 撤销
  • Ctrl+Y / Ctrl+Shift+Z 重做
  • 最大保存 50 步历史快照

表格分析 & 图表生成

智能识别数字数据，自动计算总分、排名并生成统计报告：

  • 分析表格：展示总分最高/最低 + 降序排名
  • 按合计排序并替换原表格
  • 一键生成柱状图（需图表模块配合）

支持任意多行多列，自动过滤非数值单元格。

工具栏交互与自适应

可拖拽工具栏

工具栏下方带有 ↕️ ⋮⋮ 拖拽条，支持鼠标或手指拖拽调整工具栏区域高度：

  • 调高工具栏查看更多按钮
  • 收缩为紧凑模式，自动显示滚动条
  • 移动端友好，触摸拖拽生效

每次打开编辑器，工具栏高度自适应内容，上限为视口高度 40% 。

纸张尺寸限制 & 缩放

  • 可选 A3 / A4 / A5 / Letter 纸张宽度
  • 表格放大/缩小按钮 (zoom 0.5~2倍)
  • 超出最大宽度时自动缩放表格，适配打印样式
  • 表格本身无硬性宽度限制，由内容撑开

智能格式光标感知

实时从光标所在位置读取字号并更新工具栏输入框值；若在空白单元格输入，自动包裹span并应用默认字号 (16px)，避免格式丢失。

  • 全局默认字号可动态更新
  • 支持混合内联样式与document.execCommand
  • 光标切换时字号输入框实时同步
集成方式与设计亮点

Quill Blot 嵌入

基于 Quill 的自定义 Blot（CustomTableBlot），将表格作为独立嵌入式模块，支持双击编辑、保留原始HTML。

  • 注册自定义标签 div.ql-table-container
  • 表格内容与富文本无缝混合
  • 序列化为标准HTML，支持持久化

历史快照与防抖保存

每次表格变动（内容/结构）延迟保存快照，避免高频写入。撤销/重做采用替换整表DOM方式，确保一致性。

同时支持独立于 Quill 的撤销栈，不干扰编辑器自身历史。

快捷操作指南
• 对齐模式切换：📍 光标处对齐 → 仅对当前选区所在单元格应用；📊 所有单元格对齐 → 一次性作用于全表。
• 多选模式：开启后点击任意单元格即可高亮选中，支持批量操作（对齐、颜色、字号、清除）。再次点击按钮完成选择并自动退出模式。
• 单元格内左右/上下分割：将原单元格划分为两个独立编辑区，各自内容可单独编辑，中间带分隔线。
• 纸张尺寸+缩放：先选择纸张(如A4)，点击“应用纸张尺寸”，根据表格宽度自动计算缩放比例，保持横向不溢出。
• 撤销/重做：支持所有行列增删、合并拆分、文字格式变更、内容修改等操作，误操作毫秒级恢复。
快速集成（开发者参考）

在 Quill 中加载表格模块

// 1. 引入 table-module.js 文件 (全局挂载 window.TableModule)
// 2. 初始化 Quill 后调用
var quill = new Quill('#editor', { theme: 'snow' });
window.TableModule.init(quill);

// 3. 手动打开表格编辑器 (例如工具栏按钮触发)
window.TableModule.showModal(null);
// 或编辑已有表格: showModal(htmlString)

// 4. 可选: 自定义图表模块 (需与 TableModule 配合)
window.ChartModule = { showModal: function(config){ ... } };

模块完全独立，不依赖任何第三方 UI 库。工具栏全部功能均绑定原生事件，可轻松嵌入任何后台管理系统。
