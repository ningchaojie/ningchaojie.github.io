数学终端 v7.0 (符号计算引擎)
欢迎使用数学终端。符号运算由 Nerdamer 驱动，支持不定积分、方程求解、微分/代数化简及2D/3D绘图。新增 dsolve 符号求解常微分方程 (Pyodide+SymPy)。

1. 基础计算与函数

2^10 + sin(pi/4) (支持复数、矩阵，如 det([[1,2],[3,4]]))

2. 符号运算

simplify(表达式) - 化简表达式

expand(表达式) - 展开多项式

factor(表达式) - 因式分解

3. 符号求导

diff(表达式, 变量) 例: diff(x^3 * sin(x), x)

4. 不定积分

int(表达式, 变量) 例: int(x^2, x) → x^3/3

5. 定积分 (数值/符号)

int(表达式, 变量, 下限, 上限) 优先尝试符号精确值，失败则数值积分。

例: int(exp(-x^2), x, 0, 1) 或 int(x^2, x, 0, 2)

6. 方程求解

solve(方程, 变量) - 符号求解（方程可含等号，如 x^2-4=0 或 x^2-4 自动补0）

例: solve(x^2 - 4, x) → [-2,2]

nsolve(方程, 变量, 初始值) - 数值求解（牛顿法）

例: nsolve(x^3 - x - 2, x, 2)

7. 多元方程组

solve([方程1, 方程2], [变量1, 变量2])

例: solve([x+y=3, x-y=1], [x,y]) → {x:2, y:1}

8. 常微分方程 (数值)

ode(y', y0, t_end) (默认步长0.1，t从0到t_end) 例: ode(-2*t*y, 1, 3)

9. 函数绘图

plot(表达式, 变量, 下限, 上限) 例: plot(sin(x)/x, x, -10, 10)

plot3d(表达式, [x下,x上], [y下,y上]) 例: plot3d(sin(sqrt(x^2+y^2)), [-5,5], [-5,5])

10. 常微分方程符号求解 (SymPy引擎)

dsolve(微分方程, 因变量, 自变量) - 符号求解ODE，支持一阶、二阶及常系数线性等。

例: dsolve(m*diff(x,t,2)=F, x, t) → x(t) = F*t^2/(2*m) + C1*t + C2

例: dsolve(diff(y,x)=2*y, y, x) → y(x) = C1*exp(2*x)

注意：首次使用 dsolve 会加载约6-8MB的Pyodide引擎，请稍等。
