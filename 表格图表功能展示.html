📊 表格制作与分析模块

可视化编辑 · 通用数据分析 · 一键图表

✏️ 所见即所得表格设计

支持任意行列的增删，单元格内容可直接编辑（文本、数字、公式），双击表格即可重新打开设计器修改。

📈 通用数据分析

自动识别第一行为表头、第一列为行标签，其余列为数值数据。一键计算每行的“合计值”，自动找出最高/最低合计项，并支持按合计值排序替换表格。

📊 无缝生成图表

基于表格中的数值数据（如销售额、产量、成绩等），直接调用图表模块生成柱状图/饼图/折线图，并插入编辑器。

🔁 持久化与再编辑

使用自定义 Blot 保存表格 HTML 结构，插入后仍可双击表格进入编辑器，完全保留格式与数据。

🚀 快速使用流程

  • 点击工具栏「表格」按钮，打开表格设计器。
  • 使用上方按钮增加/删除行列，双击单元格编辑内容（支持任意文字、数字）。
  • 点击「分析表格」自动计算每行的合计值、最高/最低、排名，点击「按合计值排序并替换表格」可重新排列数据行。
  • 点击「生成图表」自动将当前表格的合计数据传入图表模块，生成柱状图后可进一步编辑样式再插入。
  • 点击「插入表格」将当前设计的表格写入编辑器，之后随时双击表格即可再次编辑。
💡 通用性设计：表格分析不预设“科目”“分数”等词汇，所有提示使用“合计值”“数值数据”等通用术语，适用于任何数值型表格（财务、统计、成绩等）。

📈 图表生成模块

数据模式 · 函数曲线 · 多曲线 · 缩放编辑

📋 数据列表模式

手动编辑标签、数值及每个数据点的颜色，支持柱状图、折线图、饼图、环形图，实时预览。

🧮 函数曲线模式

输入数学表达式（如 sin(x), x^2, exp(-x)），支持多条曲线独立颜色，自动采样绘制折线图。

🎨 直观颜色设置

每个数据系列（或每条函数曲线）均配有独立颜色选择器，修改即实时更新图表预览。

🖼️ 插入后可编辑

图表以自定义图片 Blot 形式存储配置信息，双击图表即可重新打开设计器，修改后原位置更新。

↔️ 完美缩放

利用 quill-image-resize-module，插入后的图表图片支持拖拽缩放，保持宽高比。

📤 从表格导入数据

表格模块中的“生成图表”按钮可自动将合计值数据传递给图表模块，实现数据无缝流转。

⚙️ 核心功能说明

  • 数据列表模式：直接编辑标签和数值，可任意增删数据点，每个数据独立颜色。
  • 函数曲线模式：设定 X 范围、采样点数，输入表达式（支持 sin, cos, tan, log, sqrt, ^ 幂运算等），可添加多条曲线，每条曲线独立颜色。
  • 图表类型切换：柱状图、折线图、饼图、环形图即时切换，所有数据实时响应。
  • 插入与编辑：点击“插入/更新图表”将预览图片及配置保存到编辑器；双击已插入的图表图片，自动打开设计器并载入原有配置，修改后可覆盖更新。
  • 图例优化：图例默认显示数据集名称（或函数表达式），并使用圆形颜色标识，清晰区分多曲线。
// 示例：函数表达式输入 sin(x) // 正弦曲线 cos(x) // 余弦曲线 x^2 - 2*x + 1 // 抛物线 exp(-x/5)*sin(x) // 衰减振荡
🔧 技术实现：图表模块不依赖任何第三方绘图库（仅使用 Chart.js 渲染），所有配置保存在图片的 data-chart-config 属性中。表达式求值使用安全的 Function 构造器，支持常见数学函数，无需额外解析器。模块通过 ChartModule.init(quill) 挂载到编辑器，并暴露 showModal(config) 供外部调用（如表格模块生成图表时自动填充数据）。

🔌 模块化集成指南

像手写模块一样简单

两个模块均采用独立文件结构，只需在 HTML 中引入样式与脚本，并在 Quill 初始化后调用对应模块的 init 方法即可。

<!-- 在 <body> 末尾引入 -->
<link rel="stylesheet" href="table-module/table-module.css">
<script src="table-module/table-module.js"></script>
<link rel="stylesheet" href="chart-module/chart-module.css">
<script src="chart-module/chart-module.js"></script>

<script>
  var quill = new Quill('#editor', { ... });
  TableModule.init(quill);
  ChartModule.init(quill);
</script>

表格模块会自动在编辑器工具栏添加「表格」按钮，图表模块添加「图表」按钮。两个模块之间可通过 ChartModule.showModal(chartConfig) 进行数据互通（例如表格生成图表功能）。

✅ 完全兼容：与公式模块、手写模块、笔记系统等无缝协作，所有插入的内容均支持撤销/重做、保存到 IndexedDB、打印 PDF 等已有功能。
